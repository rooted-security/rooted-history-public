# Rooted History

Rooted History is a product developed by Rooted Security LLC.

It is an AI-assisted family history storytelling and investigation platform
designed to help individuals turn genealogical records into meaningful
ancestor narratives. Rooted History integrates with FamilySearch to support
research and storytelling workflows.

Website: https://rootedhistory.io

This repository is intended for public information only. Production source
code is maintained in private repositories.
